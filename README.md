  # Puja Booking Backend
   
   Backend API for Shopify GraphQL order creation.
   
   ## Endpoints
   
   - `POST /api/create-draft-order` - Create draft order in Shopify
   
   ## Environment Variables
   
   - `SHOPIFY_STORE_URL` - Your Shopify store domain
   - `SHOPIFY_ADMIN_API_TOKEN` - Admin API access token
